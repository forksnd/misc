Drawing

To make interacting with MISC a bit more visual, if the result of an expression is a valid drawing it will be displayed to the right of the console above. A valid drawing is composed of an array of path elements. Each element is either a move [m x y], a line [l x y], a bezier curve [c cx cy cx2 cy2 x y] or a close path segment [z].
