Higher order maps

There are some bootstrapped map versions of standard higher order functions. foldm, mapm, filter, zipWith. The documentation for these functions provides more examples of what they can do.

  • [mapm '[1 2 3 4 5] [lambda '[v:1] '{v [+ v 1]}]]
  • [filter '[5 4 3 2 1] [lambda '[v:value] '[< v 4]]]
  • [filter '["will" "sam" "isaac" "jemima"] [lambda '[v:value] '[< [length v] 5]]]
  • [foldm {1 2 3 4 5} f:[lambda '[v:value r:1] '[* v r]] id:1]

It is also possible to do partial application using partial-app. This allows both the fixing and renaming of arguments.

  • Doubling: [[partial-app * fix:'[2:2] rename:{}] 12]
  • Reciprocal: [[partial-app / fix:'[1:1] rename:{2:1}] 12]
  • Squaring: [mapm '[1 2 3 4 5] [partial-app * fix:{} rename:'[2:1]]]
