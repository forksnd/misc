More on data

The short-hand use of curly brackets {} can be used for data where all its components are not quoted. {a b} is equivalent to '[,a ,b].

  • {3 [* 2 2] [+ 2 3] '[+ 3 3]}

Unquoting data allows drawings and data to contain the results from calculations.

  • [let '[x:100 y:100 w:50] '{{'m x y} {'l [+ x w] y} {'l [+ x w] [+ y w]} {'l x [+ y w]} '[z]} ]
